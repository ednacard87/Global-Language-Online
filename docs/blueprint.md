# **App Name**: EduCentral

## Core Features:

- Student Progress Metrics: Display key metrics such as completed courses, grades, and study time.
- Active Courses Overview: Show active courses with completion percentages and quick access links.
- Performance Charts: Visualize weekly and monthly performance using interactive charts.
- Task Management: List upcoming tasks, recent achievements, and study time analysis.
- Quick Action Buttons: Provide buttons for easy access to classes, assignment submission, and grade viewing.

## Style Guidelines:

- Primary color: Soft teal (#96E2D6), providing a friendly and modern educational atmosphere.
- Background color: Light grey (#F8F9FB) creates a clean and accessible environment.
- Accent color: Dark grey (#1A1A1A) is used for active selections and text.
- Body and headline font: 'Inter', a modern geometric sans-serif known for clear readability.
- Line art style with circular backgrounds, employing dark grey (#1A1A1A) to achieve a friendly, clean, modern and educational look.
- Masonry-lite/Dashboard Grid layout ensures information is organized in a clean, card-based layout, fitting the SoftLearning_Glass_UI.
- Subtle transition animations on card hovers to show element activation and add to user engagement.
